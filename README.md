# API Template

Professional Node.js + Express API template showcasing clean structure, authentication, validation, and unified error handling.

Project structure:

src/
  routes/
  controllers/
  services/
  models/
  middleware/
  config/

server.js
package.json
.env.example
README.md
database/schema.sql

Features:
- JWT authentication
- Password hashing with bcrypt
- Input validation with Joi
- Centralized error handling
- Postgres database (pg)

Quick start:
1. cp .env.example .env and set values
2. npm install
3. npm run dev

This is a template intended to be shared with clients to show code organization and best practices.
